Network Tools

Vendor-neutral diagnostics that run from the toolkit host or your browser.

What’s My IP? See the client address used to connect to this toolkit server. Subnet Excluder Subtract CIDR networks from one or more parent networks. Multi-Host Ping Troubleshoot reachability, latency, and packet loss across multiple hosts. Multi-SSH Run the same command sequence across multiple SSH hosts. DNS Lookup Tester Compare DNS answers and lookup latency across multiple resolvers. RADIUS Authentication Test Compare PAP, CHAP, PEAP/MSCHAPv2, or EAP-TLS authentication across RADIUS servers. Wi-Fi / LAN Speed Test Measure browser-to-toolkit latency, jitter, download, and upload on the local network. Certificate Chain Inspector Inspect the exact TLS chain a web server presents and validate trust and hostnames. SNMP Tester Validate SNMPv2c and SNMPv3 access with reusable devices, credentials, and OID collections. TCP Port Scanner Check selected TCP ports across multiple authorized hosts. NTP Tester Measure clock offset, response delay, jitter, and synchronization health. DHCP Discover Send a Discover with a custom parameter request list and inspect Offers without requesting a lease. Path MTU Tester Find the largest unfragmented packet that reaches a destination. Webhook / API Tester Send a bounded HTTP request and inspect its status, timing, headers, and body. Syslog Tools Generate test messages or listen briefly for UDP and TCP syslog traffic. Traceroute Trace up to 10 destinations with live graphical hops and text output.
